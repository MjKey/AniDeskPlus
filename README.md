# AniDesk Plus

**AniDesk Plus** — это форк десктоп-клиента **AniDesk** (для аниме-сервиса **Anixart**) с открытым исходным кодом, расширяющий базовые возможности плеера, отладки и интеграций.

> ℹ️ Данный репозиторий является форком [проекта AniDesk](https://github.com/theDesConnet/AniDesk) от **theDesConnet**. Проект развивается отдельно разработчиком **MjKey** и содержит множество новых функций, улучшений и оптизимаций.

</div>

---

## 🚀 Ключевые возможности AniDeskPlus

- 🌸 **Полноценная интеграция с Shikimori**:
  - Отображение рейтинга и отзывов с Shikimori рядом с данными Anixart.
  - Управление статусом просмотра (*Смотрю*, *В планах*, *Просмотрено* и т.д.) и количеством серий.
  - Автоматическая фоновая синхронизация просмотренных серий в аккаунте Shikimori во время просмотра в плеере.
  - Быстрая авторизация через 1-клик OAuth2 / Personal Access Tokens.

- 🐞 **Debug Mode (`--debug`) & Live Debug Console**:
  - Запуск приложения с флагом `--debug` активирует интерактивную консоль отладки реального времени.
  - Перехват сетевых HTTP/API запросов, консольных логов и JS-ошибок.

- 🎬 **Удобный видеоплеер & Быстрая перемотка**:
  - Регулируемая кнопка перемотки с удобным изменением времени колесиком мыши (вращение мыши увеличивает/уменьшает время перемотки).
  - Улучшение качества видео с помощью **Anime4K** (WebGPU).
  - Автоматический пропуск опенингов/эндингов (Kodik & AniSkip).

- 🔔 **Windows-уведомления**:
  - Системные уведомления Windows о выходе новых серий для аниме в вашем Избранном.
  - Фильтр по предпочитаемой озвучке (например, *AniLibria*, *Studio Band*).

- 📌 **Сворачивание в трей**:
  - Сворачивание приложения в системный трей Windows при закрытии окна с быстрым контекстным меню.

- ⚡ **Оптимизация и Автообновления**:
  - Ограничение памяти V8, lazy-loading изображений и автоматическая проверка релизов.

---

## 🛠️ Сборка из исходников

Для локальной сборки проекта необходим **Node.js 20+**:

```bash
git clone https://github.com/MjKey/AniDeskPlus.git
cd AniDeskPlus
npm install
```

При необходимости укажите секреты Shikimori в файле `.env` (см. [.env.example](.env.example)):
```env
SHIKIMORI_CLIENT_ID=your_client_id
SHIKIMORI_CLIENT_SECRET=your_client_secret
```

Сборка приложения:
```bash
npm run buildAndMake
```
